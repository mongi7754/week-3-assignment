CSS Basics & The Box Model

Understanding fundamental CSS concepts and how the box model works

Introduction to CSS

Cascading Style Sheets (CSS) is used to style and layout web pages. With CSS, you can control the color, font, size, spacing, and many other aspects of HTML elements.

/* This is a CSS comment */
body {
  font-family: 'Arial', sans-serif;
  background-color: #f0f0f0;
  margin: 0;
  padding: 20px;
}

The CSS Box Model

Every element in web design is a rectangular box. The CSS box model describes how these boxes are sized and spaced.

Content
Margin Border Padding Content

Box Model Components

  • Content: The actual content of the box, where text and images appear
  • Padding: Clears an area around the content (inside the border)
  • Border: A border that goes around the padding and content
  • Margin: Clears an area outside the border (transparent)
.box {
  width: 300px; /* Content width */
  height: 150px; /* Content height */
  padding: 20px; /* Space inside the border */
  border: 5px solid #4f46e5; /* Border around padding */
  margin: 30px; /* Space outside the border */
}
